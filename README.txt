# Murat - CRM System

## Описание

Murat - это CRM (система управления взаимоотношениями с клиентами), разработанная на базе Django. Проект включает функционал для управления услугами, пользователями и кампаниями, что позволяет эффективно вести учет и взаимодействие с клиентами.

## Основные функции

- **Управление услугами**: Создание, редактирование и просмотр услуг.
- **Пользовательская аутентификация**: Регистрация и вход пользователей.
- **Статистика по кампаниям**: Просмотр статистики по кампаниям и количеству клиентов.
- **Интерфейс магазина**: Главная страница магазина, где отображаются все доступные услуги.

## Запуск проекта

### Предварительные требования

1. Убедитесь, что у вас установлены следующие программы:
   - Python 3.x
   - Django
   - Git

### Клонирование репозитория

Сначала клонируйте репозиторий на свой компьютер:

```bash
git clone https://github.com/TUTBOROMA/murat.git
cd murat
Установка зависимостей
Создайте виртуальное окружение (опционально, но рекомендуется):

bash
Копировать код
python -m venv venv
Активируйте виртуальное окружение:

На Windows:
bash
Копировать код
venv\Scripts\activate
На macOS/Linux:
bash
Копировать код
source venv/bin/activate
Установите необходимые пакеты:

bash
Копировать код
pip install -r requirements.txt
Настройка базы данных
Выполните миграции для настройки базы данных:

bash
Копировать код
python manage.py migrate
Создайте суперпользователя для доступа к админке:

bash
Копировать код
python manage.py createsuperuser
Следуйте инструкциям для создания учетной записи администратора.

Запуск сервера
Запустите сервер разработки:

bash
Копировать код
python manage.py runserver
Откройте браузер и перейдите по следующему адресу:

ruby
Копировать код
http://127.0.0.1:8000/clients/shop/
Это стартовая страничка вашего магазина, где вы сможете просмотреть доступные услуги.

Как использовать
Регистрация пользователя: Перейдите по адресу http://127.0.0.1:8000/clients/register/ для создания новой учетной записи.

Вход в систему: После регистрации войдите в систему с помощью своих учетных данных.

Управление услугами:

Для создания новой услуги перейдите по адресу http://127.0.0.1:8000/clients/services/new/.
Для просмотра списка услуг используйте http://127.0.0.1:8000/clients/services/.
Статистика по кампаниям: Перейдите по адресу http://127.0.0.1:8000/clients/statistics/, чтобы просмотреть статистику по кампаниям.

Заключение
Murat - это мощная CRM-система, которая поможет вам управлять вашими клиентами и услугами. Если у вас возникли вопросы или предложения по улучшению, не стесняйтесь обращаться!
